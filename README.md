- 灯光模块

 # EV-bike Light and Voice Control

## Project Overview
This project aims to implement lighting control and voice control functions for an electric vehicle.

## Planned Features
- On/off control of vehicle lights  
- Lighting effect and mode switching  
- Voice command recognition  
- MCU-based control logic  

## Technical Focus
- Embedded systems development  
- Voice control modules  
- Hardware drivers and interfaces
